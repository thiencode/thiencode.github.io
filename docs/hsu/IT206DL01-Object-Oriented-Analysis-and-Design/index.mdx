# IT206DL01 - Object-oriented Analysis & Design

## Course Overview

This course covers the principles and practices of object-oriented analysis and design, including:

- Object-oriented programming concepts
- UML modeling and diagrams
- Design patterns
- Software architecture principles
- System analysis and design methodologies
- And more...

## Learning Objectives

By the end of this course, students will be able to:

1. Apply object-oriented principles in software design
2. Create UML diagrams for system modeling
3. Understand and apply design patterns
4. Design robust software architectures
5. Perform system analysis using OO methodologies

## Course Structure

This course is organized into weekly modules:

- [Week 1: Introduction to Object-Oriented Analysis & Design](./week1-intro-to-OOAD)

More weeks will be added as the course progresses.
