# SW206DL01 - Algorithms Analysis and Design

## Course Overview

This course covers fundamental concepts in algorithms analysis and design, including:

- Algorithm complexity analysis
- Sorting and searching algorithms
- Graph algorithms
- Dynamic programming
- Greedy algorithms
- And more...

## Learning Objectives

By the end of this course, students will be able to:

1. Analyze the time and space complexity of algorithms
2. Design efficient algorithms for various problems
3. Understand fundamental algorithm design techniques
4. Implement and optimize algorithms in practice

## Course Structure

This course is organized into weekly modules:

- [Week 1: Introduction to Algorithms](./week1-introduction)
- [Week 2: Algorithm Complexity Analysis](./week2-algorithm-complexity)

More weeks will be added as the course progresses.
